# Cursor Landing Page Recreation

This project is a recreation of the Cursor.com landing page using only HTML and CSS.

## Sections Recreated
- Top Navigation
- Hero Section
- Trusted By
- Feature Blocks
- Feature Cards
- Testimonials
- Use Cases
- Changelog
- Team/About
- Final CTA
- Footer

## Fonts
- **Inter**: Main UI text
- **JetBrains Mono**: Code snippets and technical terms

## Colors
- Background: Black (#000000) and Dark Grey (#0C0C0C)
- Text: White (#FFFFFF) and Grey (#A1A1A1)
